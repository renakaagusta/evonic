# use_skill — Load a Skill

Lazy-load a skill's SYSTEM.md knowledge and tools into the agent context.

**Usage:** `use_skill(id="skill_id")`

**When to use:**
- Before using a skill's tools (kanban, github, pinchtab, subagent, etc.)
- When you need to understand a skill's capabilities

**Common skills:** kanban, github, pinchtab, subagent, plugin_creator
